# Usage

在任意小局的牌谱页面激活此项目。对于未处理的牌谱，约需要数十秒进行处理。加载完成后右下将显示每巡目AI的决策。

Load 表记的是四家的析数据加载状态。请注意：在数据加载成功后，当前巡目与数据可能存在错位，请使用【上一盘】【下一盘】按钮对状态进行重置。
